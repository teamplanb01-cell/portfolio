# AI/ML Portfolio (Cutting-Edge React + Tailwind + Motion)

A fast, modern portfolio for AI/ML projects. Tech: **React + Vite + TailwindCSS + Framer Motion** (TypeScript).

## Quickstart
```bash
npm install
npm run dev
# open http://localhost:5173
```

## Build
```bash
npm run build && npm run preview
```
